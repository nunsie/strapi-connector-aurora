> A wrapper around strapi-connector-bookshelf extending it for use with the aurora data api.

## Install

```sh
yarn add strapi-connector-aurora
```

or

```sh
npm i strapi-connector-aurora
```

## Usage

```sh
aurora
```

## Run tests

```sh
yarn test
```
